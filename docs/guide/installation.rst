=============
Installation
=============

``pygmtools`` can be installed by the ``pip install`` command:
    ::

        pip install pygmtools


The following packages are required, and shall be automatically downloaded:

* Python >= 3.5
* requests >= 2.25.1
* scipy >= 1.4.1
* Pillow >= 7.2.0
* numpy >= 1.18.5
* easydict >= 1.7
